(Hi!)Codevald is a software development tool. It is still bare bones and already generates clean code, as per your style. This is a code generator that will help you generate mundane code layers like the dal and bll, in a blast.
Efficient code has got a pattern or a signature for that matter, and this pattern can put into a template. Your template can be composed of any language (C#,Django,VB,PHP...), you can even write a document from XML.
Assumming the business requirements are strongly object oriented, CodeVald will be deemed successful if it aids object oriented analysis and transforms this analysis into functional software modules.
We are talking of creating a prototype for a software application, at the very least. You can now focus your energies on finer details of the system. More time is dedicated to creativity and meeting special business requirements.
We are a very small team and would appreciate the contribution of visionary forces to foster the project.
CodeVald runs on Django for python. The converters are detachable python modules. To get started, get the git code!.
